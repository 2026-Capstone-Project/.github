### Calio - 말 한마디로 일정이 완성되는 AI 일정 관리 서비스

> 흩어진 약속, 할 일, 공유 일정을 자연어 기반으로 빠르게 정리하고 함께 관리하는 캘린더 서비스입니다.
<br />

## ✨ About Calio

**Calio**는 사용자가 말하듯 입력한 내용을 바탕으로  
일정과 할 일을 빠르게 정리하고 관리할 수 있도록 돕는 AI 캘린더 서비스입니다.

일정 관리는 매일 반복되는 일이지만, 실제로는 생각보다 번거롭습니다.  
약속 내용을 다시 날짜와 시간으로 나누어 입력해야 하고, 할 일과 일정은 서로 다른 공간에 흩어지며, 함께 잡은 약속은 따로 공유하고 확인해야 합니다.

Calio는 이런 과정을 줄이고,  
개인 일정부터 친구와의 공유 일정까지 하나의 캘린더 안에서 자연스럽게 관리하는 경험을 제공합니다.

<br />

## 💡 Why Calio?

| 불편함 | Calio의 해결 방식 |
| --- | --- |
| 일정 등록 과정이 번거로워요 | 자연어 입력으로 빠르게 일정을 만들 수 있어요 |
| 할 일과 일정이 따로 흩어져요 | 캘린더 중심으로 일정과 할 일을 함께 관리해요 |
| 공유 약속을 따로 확인해야 해요 | 친구와 공유한 일정을 한눈에 확인해요 |
| 반복 일정 입력이 귀찮아요 | 반복 일정과 반복 할 일을 쉽게 관리해요 |
| 장소와 메모가 여기저기 흩어져요 | 일정 안에서 장소와 메모를 함께 기록해요 |

<br />

## 🚀 Vision

Calio는 단순한 캘린더가 아니라,  
사용자의 하루를 더 적은 입력으로 정리하는 **AI 일정 비서**를 목표로 합니다.

<br />

| 앞으로의 방향 |
| --- |
| 🗣️ 사용자가 말한 내용을 더 정확하게 일정으로 변환하기 |
| 👥 개인 일정과 협업 일정을 자연스럽게 연결하기 |
| 🔁 반복되는 일정 관리 부담 줄이기 |
| 📅 하루의 할 일과 약속을 한눈에 이해할 수 있는 경험 만들기 |
